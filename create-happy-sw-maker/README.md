# create-happy-sw-maker

Scaffold a [happy-sw-maker](https://github.com/GoodRelax/happy-soft-maker) project.

## Usage

```bash
npm init happy-sw-maker my-project
cd my-project
node setup.js
```

Then write your concept in `user-order.md` and run `/full-auto-dev`.

## What it does

1. Downloads the latest happy-sw-maker template from GitHub
2. Extracts it into the specified directory
3. That's it — no dependencies installed, no config generated
